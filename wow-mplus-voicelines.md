# WoW M+ Voiceline Reference

**Purpose:** Source dataset for the WoW dungeon trivia game. Each entry = one voiceline candidate.  
**Voiceline Types:** `BOSS_INTRO` · `BOSS_COMBAT` · `BOSS_DEATH` · `BOSS_SLAY` · `NPC_RP` · `DUNGEON_INTRO`  
**Status:** ✅ Quote verified on Wowhead NPC page · ⚠️ Quote from memory — verify before use

**Rules applied:**
- No lines that explicitly name the dungeon
- No sound-only lines (hisses, roars, mechanical sounds with no words)
- Quantity over curation — you pick the final cut

---

## How to find missing sound URLs

1. Go to the NPC's Wowhead page (linked in each row)
2. Click the **Sounds** tab
3. Find the line matching the quote text
4. Note the URL: `wowhead.com/sound=XXXXXX/filename`
5. That `XXXXXX` is the sound ID — verify it downloads from the page
